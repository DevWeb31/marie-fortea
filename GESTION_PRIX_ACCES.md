# 🎯 Accès à la Gestion des Prix

## ✅ Système de Prix Dynamiques Intégré !

Le système de gestion des prix est maintenant **complètement intégré** dans votre back office.

## 🚀 Comment Accéder

### 1. **Connexion au Back Office**
- Allez sur votre site : `http://localhost:5173` (ou votre URL)
- Cliquez sur l'icône d'administrateur dans le footer (petit cercle avec un personnage)
- Ou naviguez directement vers : `/admin`

### 2. **Accès à la Gestion des Prix**
Une fois connecté au back office, vous avez **3 façons** d'accéder à la gestion des prix :

#### **Option A : Via les Onglets**
- Cliquez sur l'onglet **"Gestion des Prix"** dans la navigation principale
- Icône : 💰 (Euro)

#### **Option B : Via les Actions Rapides**
- Dans le tableau de bord, section "Actions rapides"
- Cliquez sur le bouton **"Gérer les prix"**

#### **Option C : URL Directe**
- Naviguez vers : `/admin/pricing`

## 📱 Utilisation

### **Modifier un Prix**
1. Cliquez sur le champ à modifier
2. Entrez la nouvelle valeur
3. Les prix se calculent automatiquement
4. Cliquez sur **"Sauvegarder"**

### **Tester un Tarif**
1. Utilisez le **Calculateur de Prix**
2. Sélectionnez le type de service
3. Entrez la durée et le nombre d'enfants
4. Cliquez sur **"Calculer le prix"**
5. Vérifiez le résultat avant de sauvegarder

## 🔧 Dépannage

### **Si les prix ne se mettent pas à jour**
1. Vérifiez que vous êtes connecté en tant qu'administrateur
2. Actualisez la page avec F5
3. Vérifiez la connexion internet
4. Utilisez le bouton "Actualiser" dans l'interface

### **Si vous ne voyez pas l'onglet "Gestion des Prix"**
1. Vérifiez que vous êtes sur `/admin/dashboard`
2. Déconnectez-vous et reconnectez-vous
3. Vérifiez que la migration a été appliquée
